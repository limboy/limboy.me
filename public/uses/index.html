Uses

常用的软件、硬件和服务

Services

  • Cloudflare - 网络相关的服务（如 CDN、域名、静态网站等）首选它
  • GitHub - 除了代码托管外，CI/CD 也很方便
  • BackBlaze - 静态文件托管，如图片，结合 Cloudflare 很方便
  • Plausible - Privacy friendly 的访问统计服务
  • CodeSandbox - 很方便的 online IDE，自带 Preview，用来做代码演示比较方便
  • book4you - 中英文电子书俱全
  • Craft - 笔记 App
  • Vercel - 自己的 Side Project 基本都在这里了
  • Spotify - 听音乐首选

Tech

  • React - 相比其他的前端框架，还是 React 用起来比较舒服
  • Next.js - 基于 React 的全功能开发框架
  • Zola - 基于 Rust 的静态网站生成工具，Single Binary，小巧，功能也够用
  • Tailwind CSS
  • JavaScript

IDE

  • Visual Studio Code with VIM extension

Desktop Apps

  • Chrome - 开发和浏览以 Chrome 为主
  • Safari - 一些待看的文章或网站由 Safari 统一处理，因为跟移动端结合比较方便
  • Dropbox - 对比 iCloud Drive 和 One Drive，还是 Dropbox 最顺手
  • Raycast - UI 和 UX 比 Alfred 都好些，且免费，自带了剪贴板和窗口管理功能
  • SnippetsLab - 代码片段管理工具，已作为正常笔记工具来使用
  • Stats - System Monitor，开源免费
  • Mimestream - Mac Native Gmail Client
  • Sublime Merge - Git Client made by Sublime
  • PasteNow 剪贴板管理工具
  • Dash 看技术文档很方便
  • 1Password
  • Sketch
  • NetNewsWire
  • Logseq

CLIs

  • Homebrew
  • ripgrep - Better grep
  • croc - 两台电脑间传文件很方便，如 VPS 和本机
  • fzf fuzzy finder
